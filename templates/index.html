使用说明

基本使用

  • 复制 B 站视频页面链接（如 bilibili.com/video/BVxxxxx）
  • 粘贴到输入框
  • 选择音频格式（MP3 / M4A / AAC）
  • 点击「提取音频」，等待完成后自动下载

支持的链接格式

目前支持 bilibili.com/video/BV... 格式的视频链接。

文件保存位置

音频文件会保存在程序所在目录的 downloads 文件夹中。

安装 FFmpeg

本程序依赖 FFmpeg 进行音频转码。请在系统中安装 FFmpeg：

  • Windows: 运行 winget install FFmpeg
  • 或从 FFmpeg 官网 下载

关于 BiliMusic

BiliMusic v1.0.0

从 B 站视频提取纯音频的桌面工具。

技术栈

  • Python + Flask + pywebview
  • yt-dlp（视频下载）
  • FFmpeg（音频转码）
